# Agent Instructions

## Commits

- Write [Conventional Commit](https://www.conventionalcommits.org/) messages
  (`type(scope): summary`, e.g. `feat(auth): add token refresh`).
- Keep commits atomic: one logical change per commit. No monster commits.
- Do **not** add `Co-authored-by:` trailers.

## Code style

- Write minimal HTML, JS, CSS, and template code. Don't bloat.
- Prefer the smallest change that does the job; avoid unnecessary wrappers,
  abstractions, dependencies, and boilerplate.
- Match the conventions of the surrounding code.
